## 요구사항분석

### 기능적 요구사항

**1. 독서 기록 작성**

| 기능명 | 독서 기록 작성 |
| --- | --- |
| 탭 | 독서 기록 |
| 설명 | 사용자는 도서 정보와 독서 내용을 입력 폼을 통해 직접 작성하여 저장할 수 있다. |
| 필수 입력 | 제목, 저자 |
| 선택 입력 | 출판사, 진도율, 느낀점, 기록하고 싶은 구절, 한줄평, 독서 시작일, 완독일 |
| 기본 독서 상태 | 진도율 미입력 시 0% ‘읽고 싶은 책’, 진도율 100% 입력시 ‘완독’ 처리 |
| 예외 처리 | 필수 항목 미입력 시 저장 불가 처리 및 안내 메세지 표시 |
| 우선순위 | 높음 |

**2. 독서 목록 조회 및 관리**

| 기능명 | 독서 목록 조회 및 관리 |
| --- | --- |
| 탭 | 서재 |
| 설명 | 기록된 도서를 카드 형식으로 목록화하여 보여주며, 수정 및 삭제가 가능하다. |
| 표시 항목 | 제목, 저자, 독서상태, 진도율, 최근 읽은 날짜, 별점 |
| 수정 | 모든 입력 수정 가능 |
| 삭제 | 삭제 전 확인 다이얼로그 표시 |
| 우선순위 | 높음 |

**3. 도서 검색**

| 기능명 | 도서 검색 |
| --- | --- |
| 탭 | 서재 |
| 설명 | 서재에 기록된 도서를 제목 또는 저자명으로 검색하여 필터링 할 수 있다. |
| 검색 방식 | 입력값과 제목, 저자 중 하나라도 포함되면 결과에 표시 |
| 검색 결과 없음 | 결과 없음 안내 메세지 표시 |
| 우선순위 | 높음 |

**4. 도서 정렬**

| 기능명 | 도서 정렬 |
| --- | --- |
| 탭 | 서재 |
| 설명 | 서재에 기록된 도서를 사용자가 원하는 기준으로 정렬할 수 있다. |
| 정렬 기준 | 가나다순 (제목 기준 오름차순), 날짜순 (최근 읽은 날짜 기준 내림차순, 오래된 순) |
| 기본 정렬 | 날짜순 (최신순) |
| 우선순위 | 중간 |

**5. 독서 상태 관리**

| 기능명 | 독서 상태 관리 |
| --- | --- |
| 탭 | 서재 |
| 설명 | 각 도서에 독서 상태를 지정하고 변경할 수 있다. |
| 상태 값 | 읽고 싶음 / 읽는 중 / 완독 |
| 자동 전환 | 진도율 100% 저장 시 ‘완독’으로 자동 변경 |
| 필터링 | 상태별 필터링 가능 |
| 우선순위 | 높음 |

**6. 메모 및 별점**

| 기능명 | 메모 및 별점 |
| --- | --- |
| 탭 | 독서기록 / 서재 |
| 설명 | 도서별로 느낀점, 기록하고 싶은 구절, 별점을 기록할 수 있다. |
| 느낀점. 구절 | 최대 1000자 이내 자유 텍스트 |
| 별점 | 1-5점 |
| 우선순위  | 중간 |

**7. 출석 체크**

| 기능명 | 월별 출석 체크 |
| --- | --- |
| 탭 | 통계 |
| 설명 | 독서 기록에 저장된 ‘읽은 날짜’를 기반으로 월별 달력에 독서한 날을 시각적으로 표시한다. |
| 표시 방식 | 달력 형식, 독서기록이 있는 날짜에 표시 |
| 월 이동 | 이전 / 다음 월 이동 가능 |
| 우선순위 | 중간 |

**8. 진도율 통계**

| 기능명 | 평균 진도율 통계 |
| --- | --- |
| 탭 | 통계 |
| 설명 | 전체 도서의 평균 진도율 및 상태별 도서 수를 요약하여 표시한다. |
| 제공정보 | 전체 평균 진도율 / 상태별 도서 수(읽고 싶음, 읽는 중, 완독) / 총 독서 일수 |
| 시각화  | 도넛 차트 |
| 우선순위  | 중간 |

### 비기능적 요구사항

| 요구사항 | 상세 |
| --- | --- |
| 성능 | 도서 추가/수정/삭제 작업은 즉각적으로 반영되어야 한다. (1초 이내) |
| 사용성 | 반응형 디자인을 적용하여 데스크탑 환경을 지원한다. |
| 신뢰성 | local 데이터는 명시적 삭제 전까지 브라우저 재시작 후에도 유지되어야 한다. |
| 유지보수성  | 컴포넌트 단위로 코드를 구성하여 기능 추가 및 수정이 용이해야 한다. |
| 호환성  | 크롬, 엣지 최신 버전에서 정상 동작 해야한다. |
