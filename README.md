# 📚 ReadWithMe

독서 경험을 공유하는 독서 게시판 웹 애플리케이션입니다.

## 프로젝트 개요

- **프로젝트명**: ReadWithMe
- **목적**: 독후감 및 리뷰 작성, 책 정보 공유를 통한 독서 커뮤니티 형성
- **개발 방식**: 바이브코딩 (AI 활용 개발)
- **사용 AI**: Claude (Anthropic)

## 주요 기능

- 회원가입 / 로그인
- 책 검색 및 등록
- 독후감 / 리뷰 작성
- 댓글 / 좋아요
